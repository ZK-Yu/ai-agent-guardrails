# AI Agent Guidelines

这个仓库用于集中存放给 AI Agents 使用的全局行为约束、编写准则和可复用的 `AGENTS.md` 文档。

## 目录

- `Karpathy-Inspired/AGENTS.md`
  - 一份偏 Karpathy 风格的 Agent 行为准则，强调先思考、保持简单、精确改动、目标驱动执行。
- `yzk-guideline-en/AGENTS.md`
  - 英文版通用 Agent 准则。
- `yzk-guideline-cn/AGENTS.md`
  - 中文版通用 Agent 准则。

## 用法

你可以把这些 `AGENTS.md` 直接复制到新的项目里，或者和项目自己的约束文档合并使用。

如果某个项目需要更严格或更具体的规则，建议在项目内保留一份本地 `AGENTS.md`，再以这个仓库中的版本作为基础模板。
